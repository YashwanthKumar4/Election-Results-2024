# Election-Results-2024

India Election Results 2024 – Power BI Real-Time Dashboard
📌 Project Overview

This project is a Power BI interactive dashboard that analyzes the India General Election Results 2024.
The dashboard provides insights into party performance, seat distribution, state-wise results, and candidate performance using dynamic and interactive visualizations.

The goal of this project is to demonstrate data modeling, DAX calculations, and data visualization skills using Power BI.

Analyze party-wise seat distribution
Understand state-wise election results

📊 Short Description/Purpose : India Election Results 2024 – Power BI Dashboard

This project presents an interactive Power BI dashboard analyzing the India General Election Results 2024. The goal of the project is to transform raw election data into meaningful insights using data modeling, DAX, and data visualization techniques.

The dashboard highlights party-wise seat distribution, state-wise election performance, candidate vote share, and overall election trends. Data was cleaned and transformed using Power Query, and relationships were created through data modeling to ensure accurate analysis.

Interactive visuals and filters allow users to explore election results by state, party, and constituency, helping to quickly understand political performance across the country.

This project demonstrates data cleaning, DAX calculations, dashboard design, and analytical storytelling using Power BI.

⚙️ Step-by-Step Project Process

1️⃣ Data Collection

Election data was collected from publicly available election result sources and structured into a dataset for analysis.

2️⃣ Data Cleaning (Power Query)

Data was cleaned and transformed using Power Query Editor:
Removed null and duplicate values
Standardized party names
Corrected data types
Structured the dataset for analysis

3️⃣ Data Modeling

A proper data model was created to build relationships between tables such as:
Constituency Table
Candidate Table
Party Table
State Table
This helps improve query performance and dashboard accuracy.

4️⃣ DAX Measures

Several DAX measures were created to calculate key metrics such as:
Total Votes
Total Seats Won
Party Seat Share
Vote Percentage
Winning Margin

Example:
Vote % = DIVIDE(SUM(Votes), SUM(Total Votes))

5️⃣ Dashboard Development

Interactive visualizations were created in Power BI including:
Party-wise seat distribution
State-wise election results map
Top candidates by votes
Party vote share
Constituency-level analysis
